Guía de Instalación

Sigue estos sencillos pasos para instalar MiMalla en tu dispositivo y comenzar a organizar tu malla curricular

Instalación gratuita y sin registro requerido

Windows

Windows 10 o superior•Tamaño: 45 MB

Pasos de instalación

1

Descarga el archivo

Descarga el archivo MiMalla-Windows.rar desde el botón de descarga oficial.

2

Extrae la carpeta

Haz clic derecho sobre el archivo .rar y selecciona 'Extraer aquí' o usa WinRAR/7-Zip para extraer el contenido.

3

Crea un acceso directo

Busca el archivo MiMalla.exe dentro de la carpeta extraída, haz clic derecho y selecciona 'Crear acceso directo'.

4

Ejecuta la aplicación

Haz doble clic en el acceso directo o directamente en MiMalla.exe para iniciar la aplicación.

Descargar para Windows

Versión 1.0.0 • 45 MB

Consejos útiles

  • Si Windows Defender bloquea la aplicación, selecciona 'Más información' y luego 'Ejecutar de todas formas'
  • Puedes mover el acceso directo al escritorio para un acceso más rápido
  • La primera ejecución puede tardar unos segundos en cargar

macOS

macOS 10.15 (Catalina) o superior•Tamaño: 52 MB

Pasos de instalación

1

Descarga el instalador

Descarga el archivo MiMalla.dmg desde nuestra página oficial de descargas.

2

Abre el archivo DMG

Haz doble clic en el archivo MiMalla.dmg descargado para montar la imagen de disco.

3

Arrastra a Aplicaciones

Arrastra el icono de MiMalla hacia la carpeta 'Aplicaciones' que aparece en la ventana.

4

Inicia la aplicación

Ve a tu carpeta Aplicaciones y haz doble clic en MiMalla para ejecutar la aplicación.

Descargar para macOS

Versión 1.0.0 • 52 MB

Consejos útiles

  • Si macOS bloquea la aplicación, ve a Preferencias del Sistema > Seguridad y privacidad > General y permite la ejecución
  • Puedes agregar MiMalla al Dock arrastrándola desde Aplicaciones
  • La aplicación se actualizará automáticamente cuando haya nuevas versiones disponibles

Android

Android 7.0 (API 24) o superior•Tamaño: 28 MB

Pasos de instalación

1

Descarga la APK

Descarga el archivo MiMalla.apk directamente desde nuestro sitio web oficial.

2

Habilita fuentes desconocidas

Ve a Configuración > Seguridad > Fuentes desconocidas y activa la opción para permitir instalaciones.

3

Instala la aplicación

Toca el archivo APK descargado y sigue las instrucciones en pantalla para completar la instalación.

4

Abre MiMalla

Busca el icono de MiMalla en tu pantalla de inicio o cajón de aplicaciones y tócalo para abrir.

Descargar para Android

Versión 1.0.0 • 28 MB

Consejos útiles

  • En Android 8.0+, el sistema te pedirá permiso específico para instalar desde esta fuente
  • Puedes desactivar 'Fuentes desconocidas' después de la instalación por seguridad
  • La aplicación solicitará permisos de almacenamiento para guardar tus mallas localmente

Preguntas Frecuentes

Resolvemos las dudas más comunes sobre la instalación

¿Es seguro instalar MiMalla?

Sí, MiMalla es completamente seguro. Nuestro software está libre de malware y es desarrollado siguiendo las mejores prácticas de seguridad.

¿Necesito conexión a internet?

No, MiMalla funciona completamente offline. Solo necesitas internet para descargar la aplicación inicialmente.

¿Puedo usar MiMalla en múltiples dispositivos?

Sí, puedes instalar MiMalla en todos tus dispositivos. Tus datos se guardan localmente en cada dispositivo.

¿Qué hago si tengo problemas de instalación?

Si encuentras algún problema, revisa nuestra sección de soporte o contáctanos directamente para recibir ayuda personalizada.
